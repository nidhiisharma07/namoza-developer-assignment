# Task 2 – Consultation Landing Page

### Objective

Developed a responsive landing page for OrthoNow focused on lead generation and conversion tracking.

### Features

- Responsive Design
- Consultation Form
- Call Now Button
- WhatsApp Button
- Trust Indicators
- Statistics Section
- Services Section
- Testimonials
- Footer

### GTM Events Implemented

- consultation_form_submitted
- call_now_click
- whatsapp_click

These events are pushed into the browser dataLayer using JavaScript for Google Tag Manager integration.

### Technologies Used

- HTML5
- CSS3
- JavaScript

### Deployment

The landing page has been deployed successfully using **Vercel**.
